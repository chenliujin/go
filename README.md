

# 安装

```
$ yum install -y golang
```

# 获取 package

```
$ go get github.com/gin-gonic/gin
```

# hello.go

```
package main;

import "fmt";

func main() {
	fmt.Println("hello world");
}
```

# 运行

```
$ go run hello.go
```


# 编译

```
$ go build example.go
```
